## Installation & Setup

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Run Development Server**
   ```bash
   npm run dev
   ```
   The app will open at `http://localhost:3000`

3. **Build for Production**
   ```bash
   npm run build
   ```

4. **Preview Production Build**
   ```bash
   npm run preview
   ```

## Customization

### Personal Information
Update the following files with your information:

- `src/components/Hero.jsx` - Name and titles
- `src/components/About.jsx` - Bio and stats
- `src/components/Projects.jsx` - Your projects
- `src/components/Skills.jsx` - Your skills
- `src/components/Contact.jsx` - Social links

### Styling
- Modify `tailwind.config.js` for custom colors/fonts
- Edit `src/index.css` for global styles
